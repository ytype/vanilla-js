Arrow Function은 this를 바인딩 하지 못한다. 다음은 정상적으로 작동하지만

function()을 () => 로 변경하면 작동하지 않는다.

function은 동적으로 this를 가져오지만, arrow function은 상위 스코프를 정적으로 가져오기 때문 (Lexical this)


            $("button").on('click', (event) => {
              console.log("clicked")
              $(this).remove();
          })

Arrow Function에서 this를 사용하기 위해서는 이벤트의 currentTarget를 가져와야 한다.


            $('button').on('click', (event) => {
              console.log("clicked")
              var $this = $(event.currentTarget);
              $this.remove()
          })
